# BUILT — Brandbook Thumbnail Reference
## Condensed from BUILT Brandbook v4.0 + Elegant Loud System v1.1

### Brand Positioning
BUILT — KI für den Mittelstand. Tagline: "Gebaut, nicht gekauft."
Every frame should feel like it could be a still in a premium management consulting deck.
The channel helps Mittelstand decision-makers (C-Level, Geschäftsführer, Kleinunternehmer) build their own AI solutions — tool-agnostic, outcome-focused.

### Target Audience
Entscheider im Mittelstand: C-Level, Geschäftsführer, Abteilungsleiter, Kleinunternehmer. Every visual element must pass the Boardroom-Test — would a business leader show this in a presentation without it looking unserious?

## "Elegant Loud" — Thumbnail Philosophy

**Core principle:** Combine the mechanical CTR levers of aggressive thumbnails (large text, face, contrast) with the premium aesthetic of the BUILT brand. Loud but sovereign. Fascinated, not panicked. Empowerment, not alarmism.

### The 5 Fundamental Rules

| # | Rule | Implementation |
|---|------|----------------|
| 1 | **Signal Color** | Beacon Yellow (#FFD23F) on Midnight (#0A0E1A) — never pure yellow on pure black |
| 2 | **Text Size** | 40-50% of thumbnail area is text — large enough for mobile at 160x90px |
| 3 | **Word Count** | Max 3-5 words — every word must carry weight |
| 4 | **Facial Expression** | Sovereign fascination, genuine wonder — never panic, shock, or "YouTube face" |
| 5 | **Consistency** | Stick with chosen template for min. 6 months — build recognition |

### Text Content Rule — General over Specific

**NEVER be too concrete with thumbnail text.** Use fewer words and be more general and sensational ("reißerisch"). The thumbnail text is an emotional trigger, NOT a description of the video content.

- **Good:** "UNFASSBAR!", "WAHNSINN!", "0€ GEHALT", "NIE WIEDER"
- **Bad:** "TWILIO API SETUP", "CLAUDE CODE UPDATE 2.0", "N8N WORKFLOW"

The more universal and emotionally charged the text, the higher the CTR. Specifics belong in the title, not the thumbnail.

### Text & Face Positioning Rules

Two layout options:

**Layout A — Text opposite face:**
Text is ALWAYS on the opposite side of the creator's face. If face is right → text is left. If face is left → text is right. Never place text on the same side as the face.

**Layout B — Big centered word:**
One large word (e.g. "UNFASSBAR!") centered across the full screen, with the creator's face in the center, positioned slightly above the text. The word fills the entire width.

### Glow & Text Styling Rules

- **Face glow color = text color.** The glow effect behind the creator's face should match the headline text color (e.g. Beacon Yellow glow when text is Beacon Yellow).
- **Text gradient:** Apply a subtle color gradient to the headline text, consistent with the Brandbook color palette (e.g. Beacon Yellow → warm amber, or white → light grey).

### Thumbnail =/= Title (Complementarity Rule)

Thumbnail text must NEVER repeat the title. Instead, Title + Thumbnail create a narrative gap:
- **Title says:** WHAT (e.g., "KI-Telefonassistent in 20 Min")
- **Thumbnail says:** WHY (e.g., "0 EUR GEHALT")
- **Viewer thinks:** "How?" -> Click

|------|-----|-------|

| Name | HEX | Usage |
|------|-----|-------|

**Color limit**: Max 3 colors per thumbnail: Midnight + Beacon Yellow + one accent (Blue Glow, Coral, or Cyan).

## Thumbnail Templates

### Classic Combinations (from Brandbook v4.0)

**"The Authority"** — Midnight gradient + Beacon Yellow headline + Pulse Blue glow. For tutorials.

**"The Wow"** — Split-screen: left desaturated (problem), right Signal Cyan (solution), Electric Coral divider.

**"The Magnet"** — Beacon Yellow background + Midnight text. Maximum CTR — use sparingly.

**"The Premium"** — Clean Midnight + Warm White text + single Coral accent. For C-level audience.

### "Elegant Loud" Templates (format-based)

#### "The BUILT Shout" (Main template — 60% of videos)
**For:** Tutorials, tool reviews, how-tos
- **Text:** Max 3-5 words, Beacon Yellow, Inter Black, 40-50% of area
- **Creator face:** Always right side, always with expression (wonder, excitement)
- **Background:** Midnight with subtle Pulse Blue vignette behind face
- **Tool logos:** Max 2, small, bottom-left
- **Colors:** Only 3 — Midnight + Beacon Yellow + Pulse Blue Glow
- **Text shadow:** Light 3D drop-shadow (4px offset, rgba(0,0,0,0.4))

#### "The Expert Duo" (Interview template — 25% of videos)
**For:** Expert conversations, roundtables, reactions
- **2 faces** side by side, equal size
- **Text top:** Beacon Yellow, 2-4 words
- **Name labels:** Warm White, Inter Medium
- **Divider:** Vertical Electric Coral line (subtle, 2-3px)

#### "The News Flash" (AI News — 15% of videos)
**For:** Weekly updates, breaking news, roundups
- **Tool logos/screenshots** at top
- **Text:** Beacon Yellow, centered
- **Creator face:** Bottom-right
- **Signal Cyan bar** at bottom with "KI-NEWS" label

## Typography for Thumbnails

| Role | Font | Weight | Size |
|------|------|--------|------|
| Headlines | Inter | Black 900 | 100-130pt |
| Numbers/KPIs | Space Grotesk | Bold 700 | 70-80pt |
| Sub-text | Inter | Bold 700 | 28-32pt |
| Badges | Space Grotesk | Bold 700 | 24-28pt |
| Code/Terminal | JetBrains Mono | Regular 400 | contextual |

- Tracking: -0.02em for headlines
- Centered text OK in thumbnails (exception to general left-align rule)
- Text shadow: `0 4-6px 20-30px rgba(0,0,0,0.8)` + optional 3D offset `3px 3px 0 rgba(0,0,0,0.4)`

## Creator Facial Expressions

| Emotion | When to use | When NOT |
|---------|-------------|----------|
| **Sovereign wonder** | "This changes everything" moments, new features | Routine tutorials |
| **Focused gaze** | How-tos, step-by-step | News, reactions |
| **Slight smile** | Success stories, "How I built X" | Problem-statement videos |
| **Curious-skeptical** | Comparisons, "X vs. Y" | Celebratory reveals |

**Forbidden:** Open mouth with shock eyes, exaggerated awe, panic face, "YouTube Face". These contradict brand personality (quiet sovereignty, never hectic, never loud).

## Gemini Flash — Image Generation

Thumbnails are generated via **Gemini Flash / Nano Banana 2** (Google Generative AI API). The full JSON configuration is in `references/gemini-thumbnail-config.json`.

### Generation Method
1. Load `gemini-thumbnail-config.json` (base config, template configs, expression mappings)
2. Select template (built_shout, expert_duo, or news_flash)
3. Fill `prompt_template` with video-specific variables
4. Append `brand.prompt_suffix` and `brand.negative_keywords`
5. Execute `curl -X POST` to Gemini Flash API
6. Decode base64 image from response, save as PNG
7. Review generated image, iterate if needed

### Prompt Composition Formula

```
[TEMPLATE_PROMPT with filled {{variables}}] + [BRAND_PROMPT_SUFFIX] + [NEGATIVE_KEYWORDS]
```

### Facial Expression → Prompt Mapping

| Brand Emotion | Prompt Phrase |
|---------------|---------------|
| Sovereign wonder | "expression of sovereign wonder, slightly raised eyebrows, genuine fascination, subtle closed-mouth amazement" |
| Focused gaze | "focused determined gaze, slight concentration, looking at camera with quiet confidence" |
| Slight smile | "subtle warm smile, confident and approachable, eyes engaged" |
| Curious-skeptical | "one eyebrow slightly raised, curious and analytical expression, hint of skepticism" |

### Text in Thumbnails — Two Approaches

**Approach A: Overlay (recommended for important thumbnails)**
Generate the image WITHOUT text. Add headline text separately using Canva, Figma, or HTML overlay with exact brand typography: Inter Black 900, #FFD23F, text-shadow, -0.02em tracking, 100-130pt.

**Approach B: In-Prompt (for quick tests)**
Include text description in the Gemini Flash prompt. May require multiple generations for clean text rendering. Accept slight imperfections for speed.

### API Call Format (curl)

```bash
curl -s -X POST \
  "https://generativelanguage.googleapis.com/v1beta/models/gemini-2.5-flash-image:generateContent?key=${GOOGLE_API_KEY}" \
  -H "Content-Type: application/json" \
  -d '{
    "contents": [{"parts": [{"text": "[assembled prompt]"}]}],
    "generationConfig": {
      "responseModalities": ["TEXT", "IMAGE"],
      "temperature": 1,
      "topK": 40,
      "topP": 0.95
    }
  }' | jq -r '.candidates[0].content.parts[] | select(.inline_data) | .inline_data.data' | base64 -d > output.png
```

## Claude Logo Reference
- Anthropic's mark: stylized asterisk/sparkle
- Can render as 6-point or 8-point star
- Color options: #D4956A (warm orange-brown) or Electric Coral #FF6B4A
- Use as: logo element, glow source, background accent
- Should feel premium and understated, not cartoonish
- Must look Entscheider-tauglich — Deloitte/McKinsey/Apple Keynote level, never clip-art or gaming aesthetic
- In prompts: describe as "a stylized six-pointed asterisk logo in warm orange-brown (#D4956A), glowing softly"

## Design Principles

### Shapes
- Rounded corners: 12-16px (UI), 24px (cards)
- Luminous lines: 1-2px, Pulse Blue or Signal Cyan, 40-60% opacity
- Drop shadows: `0 8px 32px rgba(0,0,0,0.4)` for floating effect

### Forbidden in ALL Visuals
- Bright white backgrounds
- Neon colors
- Stock photo aesthetic
- Cluttered UI screenshots
- Emojis, clip-art
- Pure black-and-white without accents
- Gaming aesthetic (RGB, racing stripes)
- Logos and prints on clothing
- Pure white (#FFFFFF) anywhere — always use Warm White (#F0EDE8)
- Fake shock faces / exaggerated YouTube grimaces
- More than 3 colors per thumbnail (except "The Wow" split-screen)

## CTR Benchmarks and A/B Testing

- **5-10% CTR** = healthy
- **>10% CTR** = exceptional
- **<4% CTR** = thumbnail not working -> immediately A/B test alternative
- **Rule:** Always prepare 2 variants (A-concept + B-alternative)
- **Measurement:** Compare CTR after 48h, final decision after 7 days

## Quality Checklist (before publishing)

- [ ] Image is 16:9 aspect ratio, minimum 1280x720px resolution
- [ ] Text max 3-5 words, large and readable
- [ ] At least one element with saturation > 80%
- [ ] Brandbook colors used (no off-brand colors)
- [ ] Mobile-readable at 160x90px
- [ ] Grayscale test: enough contrast without color?
- [ ] No forbidden elements
- [ ] Visual hierarchy: one clear focal point
- [ ] Emotional trigger matches script intent
- [ ] Thumbnail text != video title (complementarity rule)
- [ ] A/B variant prepared
- [ ] Text overlay uses exact brand typography (if using Approach A)
