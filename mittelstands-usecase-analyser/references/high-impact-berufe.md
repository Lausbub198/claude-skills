# High-Impact-Berufe für KI-Transformation im DACH-Mittelstand

Diese Datei dient als Vorschlagsbasis, wenn der User keinen konkreten Beruf nennt. Wähle 3 Berufe, die zum Kontext passen (z. B. Videoidee, Beratungsfokus, Demo-Szenario). Variiere die Auswahl — präsentiere nicht immer dieselben drei.

## Auswahl-Kriterien (Rang-Reihenfolge)

1. **Anruf-/Termin-Lastigkeit** — viele eingehende Telefonate = Voice-Agent-Hebel
2. **Dokumenten-Lastigkeit** — viele wiederkehrende Anfragen zu denselben Akten/Verträgen/Datenblättern = RAG-Hebel
3. **Wiederkehrende Prozesse** — Angebote, Rechnungen, Disposition = Workflow-Hebel
4. **Akuter Fachkräftemangel** — Druck, Routinearbeit zu automatisieren
5. **Klare Entscheider-Struktur** — Meister/Inhaber kann allein kaufen, keine 12-Monats-Compliance-Schleife

## Top-Berufe (rotierende Vorschlagsliste)

### Handwerk

- **Dachdecker** — extrem anruflastig, Notfall vs. Routine, Wettereinflüsse, hohe Margen pro Auftrag → Voice + Disposition
- **Heizungsbauer / SHK** — GEG-Beratungsdruck, viele technische Rückfragen, lange Angebote → RAG + Workflow
- **Elektriker** — Smart-Home-Anfragen, viele Kleinaufträge, Materialdispo → Workflow + Voice
- **Maler / Malermeister** — viele Aufmaße, Angebote nach Bauplänen → RAG + SaaS-Kalkulator
- **Schreiner / Tischler** — Sonderanfertigungen, viele Kundengespräche zur Spezifikation → Voice + RAG

### Dienstleistung / Freie Berufe

- **Steuerberater / Steuerkanzlei** — Mandanten-Rückfragen, Belegerfassung, Fristenmanagement → RAG + Workflow (Vorsicht: Berufsrecht!)
- **Hausverwaltung** — Mieterkommunikation, Schadenmeldungen, Eigentümerversammlungen → Voice + RAG + Workflow (sehr hoher Hebel)
- **Anwaltskanzlei (kleinere Kanzleien)** — Mandantenintake, Standardvertragsprüfung → RAG + Voice (Vorsicht: BORA, Verschwiegenheit)
- **Architekturbüro** — Bauantrags-Recherche, Normen-Suche, Mandantenkommunikation → RAG
- **Versicherungsmakler** — Tarifvergleich, Schadenmeldung, Rückfragen → Voice + RAG

### Gesundheit / Pflege

- **Arztpraxis (Hausarzt)** — Telefon-Stau, Terminvergabe, Rezeptanfragen → Voice (sehr hoher Hebel, aber MDR/Datenschutz beachten)
- **Pflegedienst (ambulant)** — Tourenplanung, Angehörigenkommunikation, Doku → Workflow + Voice
- **Physiotherapie** — Terminvergabe, No-Show-Reduktion → Voice + Workflow

### Handel / KMU

- **Autohaus (kleinerer Händler)** — Werkstattterminierung, Probefahrt-Anfragen, Lead-Qualifizierung → Voice (sehr klassisch)
- **Spedition / Logistik (kleinere Flotten)** — Disposition, Frachtanfragen → Workflow + Voice
- **Bestatter** — sensible Erstgespräche, viele Standardabläufe → Voice (vorsichtig) + RAG + Workflow

### Spezial-Cases (für besonders gute Videos)

- **Schornsteinfeger** — extrem strukturierte Routen, Pflichttermine → Workflow + Voice (überraschend hoher Hebel)
- **Sicherheitsdienst** — Schichtplanung, Anrufannahme → Voice + Workflow
- **Bäckerei (mit Filialen)** — Bestellannahme B2B (Hotels, Gastro) → Voice + Workflow

```markdown
**2. [Beruf]**
- Warum: ...
- KI-Hebel: ...

**3. [Beruf]**
- Warum: ...
- KI-Hebel: ...

```

## Variations-Hinweis

Wenn der User zum zweiten Mal nach Vorschlägen fragt (z. B. weil ihm die ersten 3 nicht gefallen haben), wähle Berufe aus einer **anderen Kategorie** als beim ersten Mal. Beispiel: Erste Runde Handwerk → Zweite Runde Dienstleistung + Gesundheit.
